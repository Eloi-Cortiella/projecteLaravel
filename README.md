
# Projecte Laravel

Benvingut al projecte Laravel de CodiDAM.

Aquest projecte és una aplicació web construïda amb **Laravel 12**, utilitzant les tecnologies més modernes de l'ecosistema Laravel:

- **Frontend reactiu** amb Livewire 4 i Flux UI
- **Estils** amb Tailwind CSS v4
- **Autenticació** amb Laravel Fortify
- **Testing** amb Pest 4
- **Base de dades** amb Eloquent ORM
